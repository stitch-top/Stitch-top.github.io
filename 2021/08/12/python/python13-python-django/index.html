Python Django

Django由Lawrence Journal-World于2003年发明，首次向公众发布是在2005年7月，是一个后端服务器Web框架，免费开源。

1 介绍

  • 介绍
    • 遵循MVT设计模式(提供网站中浏览不同页面的方法，urls.py)
      • 模型：要呈现的数据，通常是来自数据库的数据，models.py。
      • 视图：根据用户请求返回相关模板和内容的请求处理程序，views.py。
      • 模板：包含网页布局的文本文件，及如何显示数据的逻辑，templates。
    • 安装并创建Django Web应用程序，并浏览器请求URL时，发生以下情况。
      • Django接收URL，检查urls.py文件，并调用与URL匹配的视图。
      • views.py文件中的视图检查模型，模型从models.py文件中导入。
      • 视图将数据发送到template目录中的指定模板，模板包含HTML和Django标记。
      • 并与数据一起将完成的HTML内容返回给浏览器。

1-1 创建项目

  • 创建项目
    • 创建项目，输入命令：django-admin startproject projectname。
    • 创建完成后，进入目录中运行项目：python manage.py runserver。
    • 检查INSTALLED_APPS设置，为应用创建所需数据表：python manage.py migrate。

1-2 应用程序

  • 应用程序
    • 项目中具有特定含义的Web应用程序，例如：主页、联系表单或成员数据库。
    • 在projectname目录中，输入命令：python manage.py startapp appname。

1-3 Django视图

  • Django视图
    • 接受http请求并返回http响应的Python函数，类似HTML文档。
    • 视图通常放在views.py文件中，该文件位于应用程序文件夹中。
    • 网址
      • 在应用程序文件夹中创建一个urls.py文件，输入内容。
      • 与项目同名的目录中也有一个urls.py文件，修改内容。
      • 在projectname目录中，输入命令：python manage.py runserver。
      • 打开浏览器，输入http://127.0.0.1:8000/appname/回车进行访问。
